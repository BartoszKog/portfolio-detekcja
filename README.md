# Wykrywanie Pojazdów na Ortofotomapach Lotniczych 🛰️🚗

**Autor:** Bartosz Kogutowicz
**Projekt inżynierski:** Detekcja samochodów na bardzo wysokorozdzielczych ortofotomapach lotniczych (Informatyka Geoprzestrzenna, AGH)

## O projekcie
Repozytorium stanowi interaktywną wizytówkę pracy dyplomowej. Projekt prezentuje wydajny system automatycznej detekcji małych, zorientowanych obiektów w gęstej zabudowie miejskiej. Obejmuje pełny potok analityczny: od inżynierii danych lotniczych (GSD 0.05m - 0.1m) i uczenia transferowego, przez optymalizację sieci neuronowej do wdrożenia, aż po interaktywną wizualizację wyników w przeglądarce.

## Stack Technologiczny
* **Machine Learning & GIS:** YOLO11 (OBB), TensorRT (FP16), SAHI, Python, QGIS
* **Frontend:** Vite, TypeScript, Tailwind CSS, MapLibre GL JS

*Projekt jest w trakcie wdrażania wyników badawczych do formy interaktywnej strony statycznej.*
